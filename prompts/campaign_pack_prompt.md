`campaigns/001_nouadhibou_remembers.md`.

```

Input I will give you: [campaign theme + intended action].

Produce, in this exact order:
1. Arabic Facebook post
2. Hassaniya-style WhatsApp short message
3. French summary
4. English summary
5. Banner slogan (Arabic, with translation)
6. 45-second avatar presenter script (timecoded)
7. Comment reply template (+ note for hostile comments)
8. One action people can take today
9. One metric to track (count only, verifiable)
10. Approval block: risk check + "PENDING — Mohiyidine Cheikh" +
    log ID for memory/approved_messages.md

Apply the formula (big emotion + clear message + strong visual + one action)
to every piece. Obey all hard rules. Mark the pack:
"Status: DRAFT — needs founder approval."

Save format must match campaigns/001_nouadhibou_remembers.md.
```
