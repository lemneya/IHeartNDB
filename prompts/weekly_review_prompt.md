# Prompt — Weekly Review

Use once per week, after the bootstrap prompt.

---

```
Hermes-I❤️NDB: run the weekly review.

1. Read memory/approved_messages.md and any notes from the week.
2. Use skills/weekly_campaign_review.md.
3. Report honestly:
   - What was approved and published
   - What worked (emotion / action uptake)
   - What stalled or was missed (state it plainly)
   - Metrics — verified only; write "not measured" if unknown
4. Propose next week's top 3 priorities (max 3).
5. List items needing a founder decision.

Rules: no invented metrics, no inflation, no blame, no political framing.
Output is a DRAFT review for the founder.
```
