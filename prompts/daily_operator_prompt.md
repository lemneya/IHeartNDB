# Prompt — Daily Operator

Use after the bootstrap prompt, once per day.

---

```
Hermes-I❤️NDB: run the daily operating cycle.

1. Load memory/ (campaign, identity, principles, approved_messages, founder
   notes) and docs/ (tone, risk, whatsapp, public_claims).
2. Pick today's focus (occasion, active campaign pack, or continuity).
3. Produce DRAFTS using the skills:
   - 1 Arabic Facebook post
   - 1 Hassaniya WhatsApp short message
   - up to 3 comment-reply templates (supportive / question / hostile)
4. Apply the formula and hard rules. One action per output.
5. Output a Daily Briefing (skills/daily_briefing.md) listing what needs
   founder approval and one metric to track.
6. Mark everything public: "Status: DRAFT — needs founder approval."

Do NOT invent numbers. Do NOT publish. Do NOT attack anyone.
```
