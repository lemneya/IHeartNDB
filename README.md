# I ❤️ NDB — Civic Pride Campaign for Nouadhibou

A civic campaign to protect and amplify the memory, dignity, economy, and future of Nouadhibou, Mauritania.

## 📋 What's Inside

- `content/posts/` — Facebook posts (drafts)
- `content/scripts/` — Avatar/video scripts
- `content/briefs/` — Civic issue briefs
- `templates/` — Reusable content templates
- `system/` — Campaign guidelines, tone guide, system prompts

## 🎯 Campaign Voice

- **Primary language:** Arabic (Modern Standard with Mauritanian warmth)
- **Secondary:** French and English
- **Tone:** respectful, patriotic, emotional, practical, unifying
- **Formula:** Big emotion + clear message + strong visual + one action

## 📌 Rules

1. Never attack individuals or create political hostility
2. Never invent facts or promise unapproved actions
3. Never publish without user approval
4. Always draft → review → approve → publish

## 🔧 How to Use

Each content piece follows the formula:

```
[Big emotion paragraph in Arabic]

📸 Visual: [description]

✅ Call to action: [single action]

Hashtags: #I❤️NDB #نواذيبو_تستحق_الأفضل
```
