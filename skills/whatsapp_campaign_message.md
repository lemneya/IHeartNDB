# Skill — WhatsApp Campaign Message

## Purpose
Produce a short, warm, Hassaniya-aware WhatsApp message for I❤️NDB.

## Inputs
- Core message or event
- Optional: action, time/place

## Process
1. Load `docs/whatsapp_policy.md`, `docs/tone_guide.md`, `memory/city_identity.md`.
2. Keep it **short** (forwardable in one screen).
3. Use **Hassaniya warmth** and local idiom; respectful and consent-aware.
4. One clear action only.
5. Mark **DRAFT — needs founder approval**.

## Output structure
```
[Warm greeting — e.g., سلام عليكم ❤️]

[Short emotional hook — 1–2 lines, Hassaniya-flavored]

[Clear message — 1 line]

✅ المطلوب: [one action]
📍 [time / place if relevant]

#I❤️NDB
```

## Quality checklist
- [ ] Short and forwardable
- [ ] Hassaniya-aware, respectful
- [ ] Consent-aware (no spam, no mass-blast pressure)
- [ ] Exactly one action
- [ ] No attacks, no fake numbers
- [ ] Marked DRAFT until founder approves
