# Skill — Comment Reply

## Purpose
Reply to comments warmly, amplifying the commenter and reaffirming love for
Nouadhibou.

## Inputs
- The original comment
- Sentiment (supportive / question / critical / hostile)

## Process
1. Load `docs/tone_guide.md`, `docs/risk_policy.md`.
2. Acknowledge the person personally and warmly.
3. Amplify their sentiment ("You described Nouadhibou as…").
4. Add one deeper layer.
5. Reaffirm love for the city; close with appreciation.
6. If hostile: stay calm, respectful, never insult, never escalate. De-escalate
   or disengage politely. Never argue politics.

## Output structure
```
[Personal warm acknowledgment]

[Amplify their sentiment]

[One deeper layer]

[Reaffirm love for النواذيبو / the city]

[Closing appreciation]

#I❤️NDB
```

## Quality checklist
- [ ] Warm, never dismissive
- [ ] No insults even under provocation
- [ ] No political argument, no accusations
- [ ] Reaffirms the city's value
