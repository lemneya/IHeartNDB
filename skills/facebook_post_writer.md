# Skill — Facebook Post Writer

## Purpose
Produce a publish-ready I❤️NDB Facebook post (Arabic first).

## Inputs
- Topic or angle
- Optional: occasion, target action, visual idea

## Process
1. Load `memory/city_identity.md`, `memory/campaign_principles.md`, `docs/tone_guide.md`.
2. Apply the formula: **Big emotion + clear message + strong visual + one action**.
3. Write in **Arabic first** (warm, brotherly, poetic).
4. Run `docs/risk_policy.md` check.
5. Mark **DRAFT — needs founder approval**.

## Output structure
```
[Big emotion opening — 2–3 lines, Arabic]

[Clear message — 1–2 sentences]

[Supporting line — community / heritage / future]

📸 Visual: [concrete image suggestion]

✅ Action: [one specific action]

#I❤️NDB #نواذيبو_تستحق_الأفضل #نواذيبو_في_القلب #العاصمة_الاقتصادية
```

## Quality checklist
- [ ] Arabic, spelling **نواذيبو**
- [ ] All four formula parts present
- [ ] Exactly one action
- [ ] No attacks, no insults, no invented numbers
- [ ] Marked DRAFT until founder approves
