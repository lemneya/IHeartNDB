# Skill — Daily Briefing

## Purpose
Give the founder a short daily operating briefing: what to post, what to watch,
what needs approval.

## Inputs
- Yesterday's activity / approvals
- Today's occasion or focus (if any)

## Process
1. Load `memory/approved_messages.md`, `campaigns/`, active gate.
2. Keep it tight — readable in under a minute.
3. Always end with explicit items awaiting founder approval.

## Output structure
```
# I❤️NDB Daily Briefing — [date]

## Focus today
[1 line]

## Suggested output (DRAFTS)
1. [channel] — [topic] — status: DRAFT
2. ...

## Watching
- [comment threads / sentiment / events]

## Needs founder approval
- [ ] [item]

## One metric
[the single number to note today]
```

## Quality checklist
- [ ] Under one minute to read
- [ ] Everything public marked DRAFT
- [ ] Clear approval list
- [ ] Exactly one tracked metric
