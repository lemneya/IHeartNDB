# Skill — Avatar Presenter Script

## Purpose
Write a short spoken script (typically 30–60s) for an avatar presenter video.

## Inputs
- Message / campaign
- Duration (default 45s)
- Language (Arabic default; Hassaniya flavor for spoken warmth)

## Process
1. Load `docs/tone_guide.md`, `memory/city_identity.md`.
2. Open with **big emotion** in the first 5 seconds.
3. Deliver **one clear message**.
4. Pair each beat with a **strong visual**.
5. Close on the slogan and **one action**.
6. Mark **DRAFT — needs founder approval**.

## Output structure
```
**Title:** [title]
**Duration:** [seconds]
**Language:** Arabic (Hassaniya warmth)

[0:00–0:05] Visual: […] | VO: [emotional hook]
[0:05–0:20] Visual: […] | VO: [clear message]
[0:20–0:35] Visual: […] | VO: [memory / pride layer]
[0:35–0:45] Visual: [I❤️NDB logo + انواذيبو تستحق الأفضل] | VO: [one action]

✅ On-screen action: [single action]
Status: DRAFT — needs founder approval
```

## Quality checklist
- [ ] Emotion in first 5s
- [ ] One message only
- [ ] Visual cue per beat
- [ ] Ends on slogan + one action
